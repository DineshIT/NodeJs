This is the first simpe node application without any framework.

To run the app, 
Step 1: clone/fork the repository.
Step 2: Run the "npm install" command to create the package-lock.json file.
Step 3: Run the app by using the below commands
    => node app.js
        (or)
    => npm start [start command value has been configured in the package.json file itself]
Step 4: Load the URL {"http://127.0.0.1:8075"} for getting the json response from the server in browser.
Step 5: In server log, you would able to retrieve the header value of the cient browser from where it is being called.
